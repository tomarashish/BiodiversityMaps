biodiversityMaps

biodiversity visualization framework using D3.js and leaflet.js

biodiversityMaps is a leaflet.js based biodiversity visualization web application for creating biodiversity marker. It makes easy to create mapping of geographical markers on web and allows to visualize data with different marker types.
Goal of visualization: Representation of biodiversity data over a geographical map.
